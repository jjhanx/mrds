# 첫 화면 히어로 이미지 가이드

## 권장 이미지 크기

| 용도 | 권장 크기 | 비율 | 비고 |
|------|-----------|------|------|
| **최적** | **1920 × 640** | 3:1 | 대부분 화면에서 잘 맞음 |
| 대안 | 1600 × 533 | 3:1 | 더 가벼운 파일 |
| 대안 | 1920 × 800 | 12:5 | 세로로 더 긴 이미지 |

- **최소 권장**: 가로 1200px 이상
- **파일 형식**: JPG 또는 WebP (용량 줄이기 좋음)
- **하단이 어두운 이미지**를 추천 (텍스트 "미래도시 함께 부르는 하모니"가 잘 보이도록)

---

## 이미지 교체 방법

### 방법 1: 파일로 교체 (권장)

1. 합창 관련 이미지를 **1920×640** 정도로 리사이즈
2. `hero.jpg`로 저장
3. `public/` 폴더에 복사
4. `src/app/page.tsx`에서 `HERO_IMAGE` 값을 `"/hero.jpg"`로 수정

```typescript
const HERO_IMAGE = "/hero.jpg";
```

### 방법 2: 환경 변수로 지정

`.env`에 추가:
```
NEXT_PUBLIC_HERO_IMAGE=/hero.jpg
```

또는 외부 URL:
```
NEXT_PUBLIC_HERO_IMAGE=https://example.com/your-choir-image.jpg
```

---

## 현재 표시 영역

- **모바일**: 높이 320px, 가로 전체
- **PC**: 높이 400px, 가로 전체
- `object-cover`로 비율 유지하며 꽉 채움 (잘리는 부분 생길 수 있음)
